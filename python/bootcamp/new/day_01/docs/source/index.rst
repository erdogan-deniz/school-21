python/bootcamp/new/day_01 — API reference
==========================================

Day 01 of the **new** Python Bootcamp track at School 21. The day
covers list / dict / set / tuple basics, file I/O, and small data
models built around them: a Robot, a Vector / Matrix pair, a
Polynomial, a Pascal triangle generator, and a Machine state.

Each exercise (``ex_01`` … ``ex_10``) is a runnable script that
exercises one of the models or one of the ``utils`` helpers.
